# inbox/

Unstructured capture. Where things land before they know where they go.

**When to use:**
- You had an idea in a meeting — dump it here, process later
- You read an article worth remembering — drop the link + notes here
- You're not sure if something is a project, reference, or one-off — park it here

**Processing rule:** empty the inbox at least weekly. Each item either:
- Becomes a project file → move to `projects/`
- Becomes a permanent reference → move to `references/`
- Gets scheduled → becomes a task in a `daily/` entry
- Is no longer useful → delete

The "GTD inbox" pattern — the only job of this folder is to give you one place to catch anything before it falls through cracks.
