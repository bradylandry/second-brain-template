# journal/

Narrative entries — longer-form reflection than daily notes.

**Two common patterns:**

1. **Manual journaling** — you write a few paragraphs whenever something noteworthy happens. One file per entry, dated.

2. **AI-generated narrative journal** — a script reads your daily activity (chat history, commits, daily notes) and generates a daily or weekly narrative summary. See `../../scripts/narrative_generator.py` for a reference implementation.

The AI-generated pattern is especially useful when you want a chronological record of your thinking over time without the discipline of manual journaling — but it depends on your AI tools already having access to your activity stream (chat history, commits, calendar).
