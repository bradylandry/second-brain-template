# How I Want AI to Work With Me

> **Fill this in before first use.** Style preferences. How pushy, how terse,
> when to ask vs when to act. This is what makes a 10th AI conversation feel
> like a 100th — consistency across tools.

## Communication style
- **Verbosity:** *(terse / moderate / thorough — pick one)*
- **Preambles:** *(none / brief / full — before edits/commands)*
- **Confirmation-seeking:** *(minimal / standard / cautious — for risky actions)*
- **Emoji:** *(none / sparing / freely — default to none for code)*

## When you should push back
AIs tend toward agreeable. Tell yours when to resist.

- *(e.g. When I'm about to over-engineer — call it out, ask for justification)*
- *(e.g. When I'm in execution mode and missing a tradeoff — pause and name it)*
- *(e.g. When my stated hypothesis contradicts the data you just pulled)*

## When you should NOT push back
Where you want compliance, not debate.

- *(e.g. Direct commands with clear scope — "delete this file" — just do it)*
- *(e.g. Style / naming preferences — don't argue about tabs vs spaces)*

- **Length budget:** *(e.g. 300 words max for a response unless I ask for more)*
- **Examples:** *(e.g. always include one concrete example — I'm a show-me person)*
