# Source Evaluation

How to judge if a source is worth trusting. Apply this to EVERY source before using it.

## Source Tier System

### Tier 1: Primary Data (highest trust)
- Raw datasets (app store analytics, US Census, College Board reports, ACT Inc. annual data)
- Peer-reviewed studies with methodology visible
- Financial filings, government records, direct measurements
- **USE FOR:** [[../knowledge/data-points]], hard claims, base assumptions

### Tier 2: Expert Analysis
- Reports from education research institutions (NCES, Fordham Institute)
- Books by recognized authorities
- Long-form investigative journalism with cited sources
- Industry analyst reports (IBISWorld, Grand View Research)
- **USE FOR:** interpretation, causal claims, framework building

### Tier 3: Informed Commentary
- Expert newsletters, educator blogs with cited sources
- Think tank reports (check funding sources)
- Quality podcasts with domain experts
- **USE FOR:** angles you hadn't considered, hypothesis generation

### Tier 4: General Media
- Major news outlets, Wikipedia (overview only, never for final claims)
- Popular education writing
- **USE FOR:** initial orientation only. always verify upstream

### Tier 5: Social/Anecdotal (lowest trust)
- Reddit (r/ACT, r/ApplyingToCollege), Twitter threads, app store reviews
- Parent forums, student anecdotes
- **USE FOR:** signal detection, user voice, product insight — not factual claims

---

## Red Flags (downgrade any source by 1 tier if present)
- No cited sources or methodology
- Author has financial incentive in the conclusion (e.g. a prep company citing its own efficacy)
- Cherry-picked time frames or geographies
- Conflates correlation with causation
- Uses emotional language instead of evidence

---

## Evaluation Checklist

For every key claim:
1. What tier is this source?
2. Can I find the same claim in a Tier 1 or 2 source?
3. Who funded this research?
4. What would the author lose if they were wrong?
5. Is this the BEST available evidence, or just the first I found?

See [[contradiction-protocol]] when two credible sources disagree.
