# Lens: First Principles

Forget everything you think you know. What is fundamentally true? Rebuild from the ground up.

## Core Questions
1. What is the actual problem being solved, stated in the simplest possible terms?
2. What would this look like if we invented it from scratch today, knowing nothing about existing solutions?
3. What assumptions are baked into the current market that nobody questions?
4. What is the minimum viable version of the solution that would still work?
5. What is the single most important variable that determines success or failure?

## ACT-Specific First Principles

**Strip everything away. What is actually true?**

1. The ACT tests a fixed, well-documented set of skills. The test does not change dramatically year-to-year.

2. Score improvement requires: knowing which skills you lack → practicing those specific skills → receiving feedback → repeating.

3. Most students who prep for the ACT don't improve much. The question is: why? Is it the prep method, the time invested, or the ceiling of improvability?

4. A student studying alone with a prep book and consistent effort will improve. The question is: what does software add that a book can't?

5. The TOPS scholarship is worth ~$4,000-$8,000/year in avoided tuition. A 1-point score improvement at the 19→20 threshold is worth tens of thousands of dollars. This is an unusually high-stakes tutoring context.

6. Louisiana mandates the ACT. The user has no choice about taking the test. The question is only whether they prep.

## Rebuild From Here

Given these truths, what should the product actually be?

- A diagnostic that identifies exactly which skills are below the TOPS threshold
- Targeted practice for only those skills (not a full test prep course)
- Progress tracking tied explicitly to TOPS score thresholds
- Minimum friction: students will quit if the app feels like homework

## How to Research Through This Lens
- Ask "why" five times for every assumption
- Ignore what competitors do — ask what the problem actually requires
- Find the physics of the situation: what are the hard constraints that cannot be worked around?

## Output Format
For each finding:
- **ASSUMPTION CHALLENGED:** [what everyone takes for granted]
- **FIRST PRINCIPLE:** [what is actually true underneath it]
- **IMPLICATION:** [what should we do differently because of this?]

## Voice
Philosophical and reductive. "The product solves one problem: a student needs to score 20 on the ACT to get TOPS. Everything else is noise."
