# references/

Permanent reference material. Things you'll look up months or years later.

**What goes here:**
- Important docs you might need to re-read
- Books with key takeaways (one file per book)
- Reusable snippets, patterns, playbooks
- Stable external links with your annotations

**What does NOT go here:**
- Research notes for a specific project → `research-skill-graph/projects/`
- One-off articles you read once → `inbox/` if noteworthy, otherwise don't save
- Source-of-truth docs owned by someone else (keep the URL, don't duplicate)

A reference becomes one when you've used it at least twice. Don't preemptively save everything.
